## About
This project was built to practice QA automation skills 
using the Page Object Model pattern with C# and Playwright.

## Tech Stack used:
- C# / .NET 8
- Playwright
- NUnit

## Project Structure:
- Pages/ - Page Object Models
- Tests/ - Test classes
- Models/ - Test data (UserCredentials)
